Página Inicial

Me chamo Gabriel Filipe, tenho 18 anos, estudo na Universidade Vale do Rio Doce (UNIVALE) e faço o curso de Sistemas de Informação no terceiro período. Esse web site faz parte do trabalho da Atividade Prática Supervisionada (APS 1) para o professor Antônio José Cunha Rodrigues.

Utilizei alguns recursos do HTML 5 e do CSS 3 para está fazendo o site. Usei como editor de texto o Brackets para desenvolver o hmtl e os recursos das folhas de estilo. Não se teve qualquer recurso de framework. Usei como fonte de estudos o site w3schools. Algumas imagens foram encontradas livremente pela internet com auxilio do Google.

Foi usado para realizar o teste os navegadores Mozilla Firefox e Google Chrome. Foi feito testes na plataforma Windows 7 Ultimate / Enterprise e no Arch Linux.
